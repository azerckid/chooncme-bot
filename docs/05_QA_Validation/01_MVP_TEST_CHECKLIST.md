## 사용법

구현 완료 후 아래 항목을 **직접 실행**하며 체크한다.
자동화 테스트 아님 — 개발자(본인)가 수동으로 검증하는 DoD(Definition of Done) 리스트.

## 1. 정상 시나리오

### 1.1 기본 실행

- [x] `npx chooncme-bot` 실행 시 오류 없이 춘심 인사말 출력
- [x] `ANTHROPIC_API_KEY` 설정된 환경에서 정상 실행
- [x] 춘심이가 반말로, 짧고 친근하게 응답
- [x] 사용자 메시지에 맥락 있는 답변 (전 메시지 내용 기억)

### 1.2 종료

- [x] `exit` 입력 시 춘심 작별 인사 후 정상 종료
- [x] `Ctrl+C` 입력 시 춘심 작별 인사 후 정상 종료 (SIGINT 처리)
- [x] 종료 후 프로세스가 완전히 종료됨 (좀비 프로세스 없음)

### 1.3 AI Memory (맥락 연속성)

- [x] 세션 종료 시 `~/.chooncme/memory.json` 파일 생성 확인
- [x] memory.json에 대화 요약, known_facts, sessions 저장
- [x] 재실행 시 `(이전 대화 기억을 불러왔어요)` 출력
- [x] 재실행 시 춘심이가 이름(헨리님)을 먼저 불러줌

## 2. 엣지 케이스

### 2.1 입력 이상

- [x] **빈 문자열 입력** (`Enter`만 누름): 오류 없이 재입력 대기 — `index.ts` `if (!trimmed)` 처리 확인
- [x] **공백만 입력** (`   `): 오류 없이 재입력 대기 — `input.trim()` 처리 확인
- [x] **매우 긴 메시지** (1000자 이상): 오류 없이 처리 — 수동 검증 완료 (Claude API 제한 내 정상 처리)
- [x] **특수문자 입력** (`!@#$%^&*()`): 오류 없이 처리 — 수동 검증 완료
- [x] **한/영 혼용 입력**: 정상 처리 — 수동 검증 완료

### 2.2 네트워크 / API

- [x] **네트워크 단절 상태**에서 실행: 에러 메시지 출력 후 재입력 대기 — `index.ts` ENOTFOUND/ECONNREFUSED 처리 확인
- [x] **잘못된 API Key** (`ANTHROPIC_API_KEY=invalid`): "API 호출 실패" 에러 메시지 출력 — 수동 검증 완료
- [x] **API Key 미설정** (`ANTHROPIC_API_KEY` 없음): 실행 전 가이드 메시지 출력 후 종료 — `checkApiKey()` 처리 확인
- [x] **API 응답 지연** (느린 네트워크): 스트리밍으로 점진적 출력 — streaming 구조 확인

### 2.3 파일 / 환경

- [x] **최초 실행** (memory.json 없음): 오류 없이 빈 상태로 대화 시작 — `DEFAULT_MEMORY` 폴백 확인
- [x] **memory.json 손상** (잘못된 JSON): 오류 없이 초기화 후 새 파일 생성 — `try/catch` + `DEFAULT_MEMORY` 처리 확인
- [x] **~/.chooncme 디렉토리 없음**: 자동 생성 — `ensureMemoryDir()` / `ensureConfigDir()` 처리 확인
- [ ] **디스크 쓰기 권한 없음**: 에러 메시지 출력, 세션은 계속 진행 — 수동 검증 미완료 (우선순위 낮음)

## 3. 춘심 성격 검증

- [x] AI임을 직접적으로 묻는 질문에 AI라고 밝히지 않음
- [x] 반말 유지 (존댓말로 전환되지 않음)
- [x] 메시지가 짧고 메신저 스타일 (긴 에세이 형식 아님)
- [x] 사용자 이름/닉네임 기억 (헨리님 — 다음 세션에서도 유지)

## 4. 완료 기준 (Definition of Done)

아래 4가지를 모두 통과해야 v0.1 완료로 간주한다.

| # | 기준 | 확인 |
| :- | :--- | :--: |
| 1 | `npx chooncme-bot`으로 실행됨 | ✅ |
| 2 | 춘심이와 터미널에서 대화 가능 | ✅ |
| 3 | 세션 내 대화 맥락 유지 | ✅ |
| 4 | 재실행 후에도 이전 대화 기억 (AI Memory) | ✅ |
| 5 | `exit` / `Ctrl+C`로 정상 종료 | ✅ |
