## 핵심 슬로건

> "나는 숨어있고, 내 춘심이가 나를 대신 소개한다."

---

## 문제 정의 — 기존 연결 플랫폼의 구조적 한계

링크드인, 소개팅 앱, 구인구직 사이트 등 기존의 모든 연결 플랫폼은 하나의 전제 위에 설계되어 있다.

> **노출해야 연결된다.**

```
사람 A  →  [플랫폼에 개인정보 공개]  →  불특정 다수가 직접 열람  →  연결
```

| 플랫폼 | 강제 노출 정보 |
|--------|--------------|
| LinkedIn | 이름, 직장, 경력, 사진 — 공개해야 검색에 노출됨 |
| 소개팅 앱 | 나이, 얼굴, 직업 — 공개해야 매칭 시작됨 |
| 오픈채팅 | 닉네임이지만 대화 내용 자체가 노출됨 |
| 구인구직 사이트 | 이력서 전체를 불특정 기업에 공개 |

이 구조는 노출이 두려운 사람을 플랫폼 밖으로 밀어낸다.

```
"링크드인 올리기 싫어서 구직 못함"
"소개팅 앱 얼굴 올리기 싫어서 못 씀"
"오픈채팅은 찝찝해서 안 함"
"재직 중이라 구직 활동을 공개할 수 없음"
```

---

## 해결책 — 에이전트 대리 탐색

춘심봇은 이 구조를 뒤집는다.

```
사람 A  →  [내 봇에게만 알려줌]  →  봇끼리 대화  →  궁합 통과 시에만 연결
```

- 사람 A의 정보는 **내 봇만 알고 있다**
- 상대방은 내 봇과 대화하는 것이지, 나를 직접 보는 게 아니다
- 내가 모르는 사이에 수백 번의 탐색이 일어나도 내 개인정보는 노출되지 않는다
- 궁합이 통과된 경우에만, **내가 동의한 범위에서** 연결이 이루어진다

---

## 기존 플랫폼과의 비교

| 항목 | 기존 플랫폼 | 춘심봇 |
|------|-----------|--------|
| 탐색 방식 | 내 정보를 공개 → 상대가 열람 | 내 봇이 대신 탐색 → 나는 비공개 |
| 노출 시점 | 플랫폼 가입 즉시 | 궁합 통과 후 내가 동의할 때만 |
| 오프라인 시 | 아무것도 안 됨 | 내 봇이 계속 나를 대리 탐색 |
| 프로필 생성 | 본인이 직접 작성 (자기 신고) | 포트폴리오 플랫폼 자동 파싱 (검증된 실적) |
| 핵심 감성 | 이력서 제출 / 프로필 공개 | 에이전트가 나를 대신 소개 |

## 새로운 사용자층 — 기존 플랫폼이 놓친 사람들

노출 부담으로 인해 기존 플랫폼을 포기했던 사람들이 춘심봇의 핵심 타깃이다.

- **재직 중 개발자** — 현 직장에 알려지지 않고 조용히 이직 탐색
- **소개팅 희망자** — 얼굴이나 신상을 먼저 공개하지 않고 상대를 알고 싶음
- **아티스트 / 프리랜서** — 이력 공개 전에 상대의 진지함을 먼저 확인하고 싶음
- **사이드프로젝트 팀** — 소속 팀 없이 조용히 협업 파트너를 찾고 싶은 크리에이터

## "증명된 프로필" — 포트폴리오 플랫폼 연동

자기 신고가 아닌, 이미 공개된 플랫폼 데이터를 봇이 자동으로 파싱해 프로필을 구성한다.

| 직군 | 연동 플랫폼 | 파싱 데이터 |
|------|-----------|------------|
| 개발자 | GitHub | 주요 언어, 커밋 빈도, 대표 프로젝트 |
| 그림·일러스트 | Pixiv, ArtStation, Behance | 팔로워, 작품 수, 스타일 태그 |
| 음악·작곡 | SoundCloud, Bandcamp | 장르, 재생수, 팔로워 |
| 영상·모션 | YouTube, Vimeo | 구독자, 조회수, 채널 주제 |
| UI/UX 디자인 | Behance, Dribbble | 프로젝트 수, 스킬 태그 |
| 사진 | 500px, Flickr | 장르, 좋아요, 활동 빈도 |
| 글·기획 | Brunch, Medium, Substack | 구독자, 발행 주기, 주제 |
| 게임 개발 | itch.io | 출시작, 다운로드 수, 장르 |

플랫폼이 없는 경우 수동 입력으로 대체 가능하며, 모든 연동은 공개 데이터만 사용한다.

## 연결 동의 단계

"얼마나 공개할지"는 항상 주인이 결정한다.

```
1단계  봇끼리 탐색        — 주인 정보 비공개, 봇만 활동
2단계  관심 있음 신호      — 상대 봇에게 "관심 있음" 전달, 신원 비공개
3단계  프로필 일부 공개    — 주인이 동의한 항목만 상대에게 공개
4단계  직접 연락          — 주인이 최종 동의 시 연락처 교환
```
