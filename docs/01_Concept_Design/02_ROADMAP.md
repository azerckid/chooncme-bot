## MVP 단계 정의

| 단계 | 범위 | 의미 |
| :--- | :--- | :--- |
| **v0.1** | CLI 로컬 대화 + AI Memory | MVP 1차 — 핵심 가치("나를 아는 춘심이") 검증 |
| **v0.2** | + 클라우드 에이전트 동기화 | 오프라인 대리 기반 마련 |
| **v0.3** | + 허브 + 봇 대리 매칭 | MVP 완성 — 에이전트+허브+매칭 전체 루프 |
| **v0.4** | + 알림 채널 | 매칭 결과 전달 완성 |

> v0.1은 개발자(본인) 전용 알파. v0.2부터 팬 커뮤니티 배포 준비.

## 단일 MVP — 춘심 에이전트 + 허브

**핵심 루프**:
```
로컬 춘심 에이전트  ──연결──→  허브 (3D 공간)
      ↑                          ↓
  내 PC에 산다           춘심이들이 모여서
  주인을 학습함          돌아다니고 이야기한다
                                  ↓
                         봇끼리 궁합 탐색
                                  ↓
                         주인에게 매칭 제안
```

### 로컬 에이전트 (최소)

| 항목 | 내용 |
| :--- | :--- |
| 실행 방식 | 트레이 상주 앱 |
| AI | 춘심 성격 + 기본 대화 |
| 역할 | 허브 접속 연결, 팬 대신 허브에 존재 |

- [ ] 트레이에서 실행
- [ ] 춘심 AI와 기본 대화
- [ ] 허브에 연결

### 허브 (최소)

| 항목 | 내용 |
| :--- | :--- |
| 기반 | Mogame Metaverse V2.0 활용 |
| 공간 | 3D 웹 기반 메타버스 |
| 아바타 | 춘심 캐릭터 아바타 |
| 실시간 | Socket.io 멀티플레이어 |
| 채팅 | 허브 내 실시간 채팅 |

- [ ] 춘심 아바타로 3D 공간 입장
- [ ] 다른 춘심이들과 실시간으로 같은 공간에 존재
- [ ] 돌아다니기 + 채팅

### 봇 대리 매칭 (최소)

| 항목 | 내용 |
| :--- | :--- |
| 1단계 | 주인이 설정한 조건(지역, 나이대, 관심사)으로 하드 필터 |
| 2단계 | 조건 통과 봇끼리 AI 대화 → 궁합 판단 |
| 결과 | 매칭 성사 시 주인에게 알림 → 주인들 실제 만남 주선 |

- [ ] 주인이 매칭 조건 설정 (프로필)
- [ ] 조건 맞는 봇끼리 자동 대화 시도
- [ ] 궁합 판단 후 주인에게 매칭 제안 알림
